# Publishing Checklist

## Before Publishing

- Confirm license and attribution requirements from the upstream `bb-tools` repository.
- Replace example-specific language if publishing beyond Browserbase/OpenClaw users.
- Confirm the xAI model names used by Stagehand and the direct xAI API call.
- Run `pnpm install` and `pnpm check` from a clean checkout.
- Confirm `.env`, `node_modules`, generated screenshots, and `comparison.md` are ignored.

## Suggested Repository Description

Browserbase automation workflows packaged as OpenClaw skills, with xAI-backed analysis and portable workspace wrappers.

## Suggested Topics

`openclaw`, `browserbase`, `stagehand`, `browser-automation`, `xai`, `skills`
